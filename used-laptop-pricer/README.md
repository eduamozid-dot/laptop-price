# Used Laptop Pricer - افزونه محاسبه قیمت لپ‌تاپ دست دوم

افزونه پیشرفته وردپرس برای محاسبه قیمت لپ‌تاپ‌های دست دوم بر اساس روش Market-Based Pricing با پشتیبانی کامل از زبان فارسی و RTL.

## 🎯 ویژگی‌ها

### ✨ ویژگی‌های اصلی
- **محاسبه هوشمند قیمت**: الگوریتم Market-Based Pricing برای محاسبه دقیق قیمت
- **پشتیبانی کامل از RTL**: رابط کاربری کاملاً راست‌چین و فارسی
- **طراحی واکنش‌گرا**: سازگار با تمام دستگاه‌ها (موبایل، تبلت، دسکتاپ)
- **مدیریت کامل از پنل ادمین**: کنترل تمام جنبه‌های افزونه
- **ورود و خروجی Excel**: پشتیبانی از فایل‌های Excel برای مدیریت داده‌ها

### 🔧 ویژگی‌های فنی
- **امنیت بالا**: استفاده از WordPress Nonce و Sanitization
- **کد تمیز**: مطابق با استانداردهای وردپرس
- **قابلیت توسعه**: ساختار ماژولار و قابل گسترش
- **پشتیبانی از ترجمه**: آماده برای ترجمه به سایر زبان‌ها

## 📋 پیش‌نیازها

- WordPress 5.0 یا بالاتر
- PHP 7.4 یا بالاتر
- MySQL 5.6 یا بالاتر
- Composer (برای نصب وابستگی‌ها)

## 🚀 نصب و راه‌اندازی

### 1. نصب از طریق Composer

```bash
# در پوشه wp-content/plugins
composer create-project hoseinmos/used-laptop-pricer
```

### 2. نصب دستی

1. فایل ZIP افزونه را دانلود کنید
2. در پنل ادمین وردپرس، به بخش **افزونه‌ها > افزودن** بروید
3. فایل ZIP را آپلود و نصب کنید
4. افزونه را فعال کنید

### 3. نصب وابستگی‌ها

```bash
cd wp-content/plugins/used-laptop-pricer
composer install
```

## ⚙️ تنظیمات اولیه

### 1. تنظیمات عمومی
1. به **لپ‌تاپ پرایسر > تنظیمات** بروید
2. واحد پول و نماد پول را تنظیم کنید
3. نرخ‌های استهلاک را تعیین کنید
4. ضرایب وضعیت ظاهری را تنظیم کنید

### 2. ورود مدل‌های پایه
1. به **لپ‌تاپ پرایسر > مدیریت مدل‌ها** بروید
2. از بخش **ورود از Excel** استفاده کنید
3. یا به صورت دستی مدل‌ها را اضافه کنید

### 3. مدیریت قطعات
1. به **لپ‌تاپ پرایسر > مدیریت قطعات** بروید
2. قیمت قطعات مختلف را وارد کنید
3. انواع مختلف CPU، RAM، GPU، و Storage را تعریف کنید

### هوک‌های WordPress
افزونه از هوک‌های مختلفی پشتیبانی می‌کند:

```php
// تغییر نرخ استهلاک
add_filter('ulp_depreciation_rate', function($rate, $year) {
    return $rate * 1.1; // افزایش 10%
}, 10, 2);

// تغییر الگوریتم محاسبه
add_filter('ulp_price_calculation', function($price, $data) {
    // محاسبه سفارشی
    return $price;
}, 10, 2);
```

## 🔧 API و توسعه

### کلاس‌های اصلی
- `ULP_Price_Calculator`: محاسبه قیمت
- `ULP_Admin_Settings`: تنظیمات ادمین
- `ULP_Model_Manager`: مدیریت مدل‌ها
- `ULP_Parts_Manager`: مدیریت قطعات

### توابع کمکی
```php
// دریافت تنظیمات
$settings = ulp_get_settings();

// محاسبه قیمت
$calculator = new ULP_Price_Calculator();
$result = $calculator->calculate_price($brand, $model, $year, $condition, $cpu, $ram, $gpu, $storage);

// فرمت کردن قیمت
$formatted_price = ulp_format_price($price);
```
