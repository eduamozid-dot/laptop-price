# راهنمای نصب و راه‌اندازی افزونه لپ‌تاپ پرایسر

## 📋 پیش‌نیازها

### سیستم مورد نیاز
- **وردپرس**: نسخه 5.0 یا بالاتر
- **PHP**: نسخه 7.4 یا بالاتر
- **MySQL**: نسخه 5.6 یا بالاتر
- **حافظه**: حداقل 128MB RAM
- **فضای دیسک**: حداقل 50MB فضای آزاد

### افزونه‌های مورد نیاز
- **Composer**: برای نصب وابستگی‌ها (اختیاری)

## 🚀 مراحل نصب

### مرحله 1: دانلود و نصب افزونه

#### روش اول: نصب از طریق پنل ادمین
1. وارد پنل ادمین وردپرس شوید
2. به بخش **افزونه‌ها** > **افزودن** بروید
3. روی **آپلود افزونه** کلیک کنید
4. فایل ZIP افزونه را انتخاب کنید
5. روی **نصب** کلیک کنید
6. پس از نصب، روی **فعال‌سازی** کلیک کنید

#### روش دوم: نصب دستی
1. فایل ZIP افزونه را دانلود کنید
2. محتوای فایل را در پوشه `wp-content/plugins/` استخراج کنید
3. نام پوشه باید `used-laptop-pricer` باشد
4. از طریق پنل ادمین، افزونه را فعال کنید

### مرحله 2: نصب وابستگی‌ها

#### نصب Composer (اگر نصب نیست)
```bash
# دانلود Composer
curl -sS https://getcomposer.org/installer | php
mv composer.phar /usr/local/bin/composer
```

#### نصب وابستگی‌های افزونه
```bash
# رفتن به پوشه افزونه
cd wp-content/plugins/used-laptop-pricer

# نصب وابستگی‌ها
composer install --no-dev --optimize-autoloader
```

### مرحله 3: تنظیمات اولیه

1. **فعال‌سازی افزونه**
   - به بخش **افزونه‌ها** بروید
   - افزونه "لپ‌تاپ پرایسر" را پیدا کنید
   - روی **فعال‌سازی** کلیک کنید

2. **دسترسی به منوی افزونه**
   - منوی جدید "لپ‌تاپ پرایسر" در پنل ادمین ظاهر می‌شود
   - روی آن کلیک کنید تا وارد داشبورد شوید

## ⚙️ تنظیمات اولیه

### مرحله 1: تنظیمات عمومی
1. به منوی **لپ‌تاپ پرایسر** > **تنظیمات** بروید
2. **نرخ استهلاک** را تنظیم کنید:
   - سال اول: 30%
   - سال دوم: 15%
   - سال‌های بعدی: 10%
3. **ضرایب وضعیت ظاهری** را بررسی کنید
4. **واحد پول** را تنظیم کنید (پیش‌فرض: تومان)
5. روی **ذخیره تنظیمات** کلیک کنید

### مرحله 2: افزودن مدل‌های پایه

#### روش اول: افزودن دستی
1. به منوی **مدل‌های پایه** بروید
2. فرم افزودن مدل جدید را پر کنید:
   - برند (مثل Dell، Apple، Lenovo)
   - مدل (مثل XPS 13، MacBook Pro)
   - سال عرضه
   - قیمت پایه
   - مشخصات پایه (CPU، RAM، GPU، Storage)
3. روی **افزودن** کلیک کنید

#### روش دوم: آپلود فایل Excel
1. فایل Excel با فرمت مناسب آماده کنید
2. ستون‌های مورد نیاز:
   - برند
   - مدل
   - سال عرضه
   - قیمت پایه
   - CPU پایه
   - RAM پایه
   - GPU پایه
   - Storage پایه
3. فایل را آپلود کنید
4. روی **آپلود** کلیک کنید

### مرحله 3: افزودن قطعات

#### روش اول: افزودن دستی
1. به منوی **قطعات** بروید
2. برای هر نوع قطعه (CPU، RAM، GPU، Storage) قیمت‌ها را اضافه کنید
3. نام، مشخصات و قیمت هر قطعه را وارد کنید

#### روش دوم: استفاده از داده‌های نمونه
1. در صفحه قطعات، روی دکمه‌های **افزود نمونه** کلیک کنید
2. داده‌های نمونه به صورت خودکار اضافه می‌شوند
3. می‌توانید قیمت‌ها را بعداً ویرایش کنید

## 🔧 عیب‌یابی

### مشکلات رایج

#### خطای "فایل Excel آپلود نشد"
**راه‌حل:**
- بررسی مجوزهای پوشه `wp-content/uploads/`
- تنظیم مجوزها: `chmod 755 wp-content/uploads/`
- بررسی اندازه فایل (حداکثر 5MB)

#### خطای "مدل یافت نشد"
**راه‌حل:**
- بررسی وجود مدل در دیتابیس
- بررسی تطابق برند و مدل
- بررسی سال عرضه

#### خطای "PhpSpreadsheet یافت نشد"
**راه‌حل:**
```bash
cd wp-content/plugins/used-laptop-pricer
composer install
```

#### خطای "دسترسی غیرمجاز"
**راه‌حل:**
- بررسی نقش کاربر (باید Administrator باشد)
- بررسی تنظیمات امنیتی وردپرس

### لاگ‌گیری
```php
// فعال کردن لاگ‌گیری
define('WP_DEBUG', true);

// مشاهده لاگ‌ها
tail -f wp-content/debug.log
```

## 🔒 امنیت

### تنظیمات امنیتی توصیه شده
1. **به‌روزرسانی منظم**: همیشه از آخرین نسخه افزونه استفاده کنید
2. **پشتیبان‌گیری**: به طور منظم از دیتابیس پشتیبان تهیه کنید
3. **مجوزهای فایل**: تنظیم مجوزهای مناسب برای فایل‌ها
4. **فایروال**: استفاده از فایروال برای محافظت از سرور

### بررسی امنیت
```bash
# بررسی مجوزهای فایل‌ها
find wp-content/plugins/used-laptop-pricer -type f -exec chmod 644 {} \;
find wp-content/plugins/used-laptop-pricer -type d -exec chmod 755 {} \;

# بررسی مالکیت فایل‌ها
chown -R www-data:www-data wp-content/plugins/used-laptop-pricer
```
